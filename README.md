
An **end-to-end Business Intelligence platform** built using **Power BI, SQL, Excel, and DAX** to analyze credit card transaction and customer behavior.

This solution integrates multi-source financial datasets to deliver **real-time KPI monitoring, customer analytics, and spending insights** through interactive dashboards.

The project analyzes **46M+ transactions** and **$57M in revenue**, helping stakeholders monitor financial performance, customer behavior, and credit portfolio trends.

---

# Project Overview

This project builds a **financial analytics solution** that transforms raw transaction and customer data into actionable business insights.

The system includes:

- Automated **SQL-based ETL pipelines**
- Data transformation and modeling
- Advanced **DAX analytics**
- Interactive **Power BI dashboards**
- Weekly performance monitoring

The dashboards allow stakeholders to track:

- Revenue growth
- Transaction trends
- Customer segmentation
- Geographic spending patterns
- Interest revenue generation

# Dashboard Features

## Transaction Analysis Dashboard

Provides insights into transaction behavior and financial performance.

### Metrics Tracked

- Weekly revenue trends
- Transaction volume growth
- Spending category analysis
- Payment behavior
- Card usage distribution

### Key Insights

- **28.8% revenue growth in Week 53**
- **12.5% increase in transaction count**
- **15.2% increase in transaction value**

---

## Customer Analytics Dashboard

Provides insights into customer demographics and financial behavior.

### Metrics Analyzed

- Revenue by gender
- Card tier usage
- Geographic revenue distribution
- Interest revenue contribution
- Customer segmentation

### Key Insights

- **$31M revenue from male customers**
- **$26M revenue from female customers**
- **$8M total interest revenue**
- **93% of transactions from Blue & Silver card tiers**
- **68% of revenue from TX, NY, and CA**
